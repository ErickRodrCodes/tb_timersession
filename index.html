What is tb_timersession?

tb_timersession is a jQuery plugin that allows you to control idle timeouts of absent users. Tb_timersession allows you to generate as many control timers as you need to trigger actions as the time pass by in seconds.

The difference with other plugins is that tb_timersession allow you to control as many events you want, not just one or two! Feel free to generate timers depending on the moment you need it!

Here an example : once you do click to the button below, tb_timersession will start to create the events to fill the progressbar below.

	timers : [
		{
			time:0,
			action : function(){
				$("#progressbar1").progressbar({value:0});
			}
		},
		{
			time:1,
			action : function(){
				$("#progressbar1").progressbar({value:20});
			}
		},
		{
			time:2,
			action : function(){
				$("#progressbar1").progressbar({value:40});
			}
		},
		{
			time:3,
			action : function(){
				$("#progressbar1").progressbar({value:60});
			}
		},
		{
			time:4,
			action : function(){
				$("#progressbar1").progressbar({value:80});
			}
		},
		{
			time:5,
			action : function(){
				$("#progressbar1").progressbar({value:100});
			}
		},
	],
	keepAliveSessionUrl : "http://www.google.com/calendar/feeds/developer-calendar@google.com/public/full?alt=json",
	logoutUrl			: "./index.html",
	humanThreshold		: 1,
	keepSessionAction	: function(){
		$("#progressbar").progressbar({value:0});
	},
	showConsole			: true,
	visualConsole		: true
}

$(function(){
	$("#progressbar0").progressbar();
    $("#button-example0").click(function(){
	$(document).tb_timersession(example0);
	$(this).hide();
    });
});

 

 

Note : Once you run the script it will not stop until the last timer object is executed. The timer resets when you press a key or you do click with your mouse.

Once the time runs out, tb_timersession will load the url assigned in the parameter "logoutUrl"

  • Starting Up

tb_timersession is very easy to use and implement. At the same time, tb_timersession offers to the developer great flexibility in order to generate time events in the adequate time.

Installation

After you download the most recent copy from GitHub, get the zip file and decompress their contents, then copy the tb_timersession flavor you want (non-compressed version and minified version). The files are inside of the directory /src of the compressed file. Once you are ready, place your tb_timersession file in your script folder in your production page.

setup

The simplest way to set up your tb_timersession is to include the javascript file in the <head> of your web page, generate your parameters inside of the <script> tag and initialize the code once the page is loaded, like this:

<script type="text/javascript" src="/js/jquery.tb_timersession.js"></script>
<script>
var mytimers = {
	timers : [ 
		{
			time:10,
			action:function(){...}
		},
		{
			time:20,
			action:function(){...}
		},
		{
			time:30,
			action:function(){...}
		},
	],
	keepAliveSessionUrl : "http://mysite/keepsessionurl",
    logoutUrl			: "./index.html",
    humanThreshold		: 5,
    keepSessionAction	: function(){...}
}
</script>

Initialization

once you have done the setup of your timer event(s), you must add the following code before of the </script> tag where we wrote the timer events:

$(function(){
	$(document).tb_timersession(mytimers); 
});

With this, the timer events will be executed once the idle time matches one of the timers.

Note : DO NOT RUN MORE THAN ONE tb_timersession on your page!!! This plugin is intended to be executed once in your application.

  • Settings

Here a complete table of the settings that allows tb_timersession to behave in the way you need to manage your idle timeout events:

Argument Type Default Value Description
timers
(mandatory) 		Array[objects] [{
time : 10 ,
action : function(){ console.log('execution!')
}
}]

timers will make an array of objects who will be the events you want to trigger.

Every object contains two properties:

  • time (integer/mandatory) : Time in seconds since the idle event starts.
  • action (javascript action/optional) : the code of the action you want to execute
keepAliveSessionUrl
(mandatory) 		string(url)

"http://www.google.com"

 When a human action is detected, tb_timersession will use the value of keepAliveSessionUrl to trigger a $.get request in order to keep the session of the user alive.
logoutUrl
(mandatory) 		string(url)

"http://www.google.com"

 When the last timer is executed, tb_timersession will redirect the content of the page to the url value stored in this parameter.
humanThreshold
(optional) 		integer 10 To avoid spamming the server with any human action (with clicks or key strokes), this parameter will allow tb_timersession to wait in seconds for the next call for keepAliveSessionURL.
keepSessionAction
(optional) 		Javascript Function function(){} A function that allow to the user to execute a custom js function once the timer is reseted as consequence of the human action (with mouse clicks or key strokes)
showConsole
(optional) 		boolean false

In case you need to track what actions on the timer are being executed. It will use the console.log function to display debug messages related to the internal events of the pluggins and parameters.

Using showConsole with older browsers that does not recognise console.log as function will result in the automatic execution of the parameter visualConsole
visualConsole
(optional) 		boolean false visualConsole will add to the page a division where the events can be seen directly through the page. This will allow you to inspect the elements you store in your timers array. This is also a trigger boolean action for fallback browsers that does not recognize console.log function if showConsole is true.

// Example 4. Complete interaction using jQuery UI, and custom action per each event:
var blinker;
var forceCountDown = 59;
var countDownInterval;
var example4 = {
	timers : [
		{
			time:0,
			action : function(){
				$(".sessionTimer").removeClass("metroui-icon-signal0 metroui-icon-signal1 metroui-icon-signal2 metroui-icon-signal3").addClass("metroui-icon-signal3");
			}
		},
		{ 
			// trigger 75% of the action
			time : 10,
			action : function(){
				$(".sessionTimer").removeClass("metroui-icon-signal3").addClass("metroui-icon-signal2");
			}
		},
		{
			// trigger 50% of the action
			time : 20,
			action : function(){
				$(".sessionTimer").removeClass("metroui-icon-signal2").addClass("metroui-icon-signal1");
			}
		}, 
		{
			// trigger 25% of the action
			time : 30,
			action : function(){
				blinker = setInterval(function(){
					$(".sessionMarker").toggleClass('markToggle');
					$(".sessionTimer").toggleClass('markToggleIcon');

				},500);
				$(".sessionTimer").removeClass("metroui-icon-signal1").addClass("metroui-icon-signal0");
			}
		},
		{
			// trigger 0% of the action
			time : 40,
			action : function(){
				template = "
For security purposes You will be signed out automatically in "
                +"60 seconds due to inactivity
Close this dialog or "
				+"press any key to keep your session alive.
";
				if($("#modal_pop").length == 0){
					$(template).dialog({
						buttons : {
							"Hey! I'm still here!":function(){
								$(this).dialog('close');
								forceCountDown = 59;
								clearInterval(countDownInterval);
								$("#modal_pop").dialog('close');
								$("#modal_pop").remove();
							}
						},
						title:'Your Session seems inactive!',
						modal:true,
						open:function(){
							resizeDialog();
							
						}
					});
					resizeDialog();
					countDownInterval = setInterval(function(){
						$("#timer").text(forceCountDown);
						forceCountDown--;
					},1000);
				}
			}
		},
		{
			// trigger logout action
			time : 100
		}
	],
	keepAliveUrl		: "http://google.com",
	logoutUrl			: "./index.html",
	keepSessionAction	: function(){
		clearInterval(blinker);
		$(".sessionMarker").removeClass('markToggle');
		$(".sessionTimer").removeClass('markToggleIcon');
		
	},
	showConsole			: true,
	visualConsole		: true
}

$(function(){
	$("#button-example4").click(function(){
		$(document).tb_timersession(example4);
$(this).hide();
	});
